# Covid-19-Data-analysis-in-python
Exploratory data analysis of COVID-19 datasets using Python. Includes data cleaning, visualization, and trend analysis with Pandas, Matplotlib, and Seaborn.
This project focuses on analyzing COVID-19 data using Python to understand the spread, trends, and patterns of confirmed cases, deaths, and recoveries across different countries and dates.
The goal of this project is to practice data cleaning, exploratory data analysis (EDA), and data visualization using real-world datasets.
